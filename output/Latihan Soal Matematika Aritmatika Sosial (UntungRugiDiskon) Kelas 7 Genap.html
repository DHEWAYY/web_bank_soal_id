A. Pilihan Ganda

1.

Dalam aritmatika sosial, kondisi untung terjadi jika...

Lihat Pembahasan

Jawaban: B

Untung (Laba) diperoleh jika Harga Penjualan (HJ) lebih besar daripada Harga Pembelian (HB). Rumus: Untung = HJ - HB.

2.

Pak Budi membeli sepeda seharga Rp800.000,00. Kemudian sepeda tersebut dijual kembali dengan harga Rp950.000,00. Berapa keuntungan yang diperoleh Pak Budi?

Lihat Pembahasan

Jawaban: B

Diketahui: HB = Rp800.000, HJ = Rp950.000. Untung = HJ - HB Untung = 950.000 - 800.000 = Rp150.000.

3.

Seorang pedagang membeli barang seharga Rp200.000,00. Karena butuh uang cepat, ia menjualnya dengan kerugian 10%. Berapakah harga jual barang tersebut?

Lihat Pembahasan

Jawaban: A

Diketahui: HB = 200.000, %Rugi = 10%. Besar Rugi = 10% x 200.000 = Rp20.000. HJ = HB - Rugi HJ = 200.000 - 20.000 = Rp180.000,00.

4.

Ani membeli tas seharga Rp150.000,00 dan menjualnya dengan harga Rp180.000,00. Persentase keuntungan yang diperoleh Ani adalah...

Lihat Pembahasan

Jawaban: B

Untung = 180.000 - 150.000 = 30.000. %Untung = (Untung / HB) x 100% %Untung = (30.000 / 150.000) x 100% %Untung = 0,2 x 100% = 20%.

5.

Sebuah barang dijual dengan harga Rp220.000,00. Jika dari penjualan tersebut pedagang mendapat untung 10%, berapakah harga pembelian barang tersebut?

Lihat Pembahasan

Jawaban: B

Diketahui: HJ = 220.000, %U = 10%. Rumus Mencari HB jika diketahui %Untung: HB = (100 / (100 + %U)) x HJ HB = (100 / 110) x 220.000 HB = 20.000.000 / 110 = Rp200.000,00.

6.

Istilah berat kotor dalam aritmatika sosial disebut...

Lihat Pembahasan

Jawaban: C

Bruto adalah berat kotor (isi + kemasan). Netto adalah berat bersih (isi saja). Tara adalah berat kemasan.

7.

Sebuah karung beras bertuliskan Bruto = 50 kg dan Tara = 2%. Berapakah Netto beras tersebut?

Lihat Pembahasan

Jawaban: B

Tara = 2% x 50 kg = 0,02 x 50 = 1 kg. Netto = Bruto - Tara Netto = 50 kg - 1 kg = 49 kg.

8.

Sebuah toko memberikan diskon 20% untuk sebuah baju yang harganya Rp120.000,00. Berapakah uang yang harus dibayarkan pembeli?

Lihat Pembahasan

Jawaban: B

Besar Diskon = 20% x 120.000 = Rp24.000. Harga Bayar = Harga Awal - Diskon Harga Bayar = 120.000 - 24.000 = Rp96.000,00.

9.

Bu Susi menyimpan uang di bank sebesar Rp4.000.000,00 dengan bunga tunggal 12% per tahun. Berapakah besar bunga yang diterima Bu Susi setelah menabung selama 9 bulan?

Lihat Pembahasan

Jawaban: C

Rumus Bunga = (n/12) x %Bunga x Modal Bunga = (9/12) x 12% x 4.000.000 Bunga = (3/4) x (12/100) x 4.000.000 Bunga = 3 x 3 x 40.000 = Rp360.000,00.

10.

Pak Rudi membeli motor bekas seharga Rp5.000.000,00. Kemudian ia mengeluarkan biaya perbaikan Rp500.000,00. Jika ia ingin untung Rp1.000.000,00, maka ia harus menjual motor tersebut dengan harga...

Lihat Pembahasan

Jawaban: B

Modal Total = Harga Beli + Biaya Perbaikan Modal = 5.000.000 + 500.000 = 5.500.000. Harga Jual = Modal + Untung Harga Jual = 5.500.000 + 1.000.000 = Rp6.500.000,00.

11.

Seorang pedagang membeli 1 lusin piring seharga Rp48.000,00. Jika ia ingin mendapat untung Rp12.000,00 dari penjualan seluruh piring tersebut, maka harga jual per piring adalah...

Lihat Pembahasan

Jawaban: B

Total Harga Jual = Harga Beli + Untung Total HJ = 48.000 + 12.000 = 60.000. Harga per piring = 60.000 / 12 = Rp5.000,00.

12.

Pajak Pertambahan Nilai (PPN) biasanya dikenakan pada...

Lihat Pembahasan

Jawaban: B

PPN dikenakan pada pembelian barang kena pajak, misalnya makanan di restoran atau barang elektronik. Gaji dikenakan PPh.

13.

Ayah membeli televisi seharga Rp3.000.000,00 dan dikenakan PPN sebesar 10%. Berapa total uang yang harus dibayarkan Ayah?

Lihat Pembahasan

Jawaban: C

Besar PPN = 10% x 3.000.000 = Rp300.000. Total Bayar = Harga Barang + PPN Total = 3.000.000 + 300.000 = Rp3.300.000,00.

14.

Tabungan Andi di bank mula-mula Rp2.000.000,00. Setelah beberapa waktu, tabungannya menjadi Rp2.200.000,00. Jika bunga bank 12% per tahun, berapa lama Andi menabung?

Lihat Pembahasan

Jawaban: C

Bunga (Rp) = 2.200.000 - 2.000.000 = 200.000. Rumus: Bunga = (n/12) x %B x Modal 200.000 = (n/12) x (12/100) x 2.000.000 200.000 = n x 1/100 x 2.000.000 200.000 = n x 20.000 n = 200.000 / 20.000 = 10 bulan.

15.

Toko A memberikan diskon 30% untuk sepatu seharga Rp200.000,00. Toko B memberikan diskon 40% untuk sepatu yang sama. Selisih harga yang harus dibayar di kedua toko adalah...

Lihat Pembahasan

Jawaban: B

Selisih Diskon (%) = 40% - 30% = 10%. Selisih Harga = 10% x 200.000 = Rp20.000,00. (Atau hitung manual: Toko A bayar 140k, Toko B bayar 120k, selisih 20k).

16.

Seorang karyawan memiliki gaji Rp4.000.000,00 sebulan. Penghasilan Tidak Kena Pajak (PTKP) adalah Rp1.000.000,00. Jika PPh (Pajak Penghasilan) sebesar 10%, berapakah gaji bersih yang diterima karyawan tersebut?

Lihat Pembahasan

Jawaban: B

Penghasilan Kena Pajak (PKP) = Gaji - PTKP = 4.000.000 - 1.000.000 = 3.000.000. Besar Pajak = 10% x 3.000.000 = 300.000. Gaji Bersih = Gaji Awal - Pajak = 4.000.000 - 300.000 = Rp3.700.000,00.

17.

Bruto satu dus barang adalah 20 kg. Jika taranya 0,5 kg, maka persentase taranya adalah...

Lihat Pembahasan

Jawaban: B

%Tara = (Tara / Bruto) x 100% %Tara = (0,5 / 20) x 100% %Tara = 0,025 x 100% = 2,5%.

18.

Sebuah barang dijual dengan rugi 20%. Jika harga beli barang tersebut Rp500.000,00, maka harga jualnya adalah...

Lihat Pembahasan

Jawaban: B

Besar Rugi = 20% x 500.000 = 100.000. HJ = HB - Rugi = 500.000 - 100.000 = Rp400.000,00.

19.

Koperasi sekolah membeli 50 buku dengan harga Rp3.000,00 per buku. Jika koperasi menginginkan keuntungan total Rp50.000,00, berapa harga jual per buku?

Lihat Pembahasan

Jawaban: B

Total Modal = 50 x 3.000 = 150.000. Target Penjualan Total = Modal + Untung = 150.000 + 50.000 = 200.000. Harga Jual per buku = 200.000 / 50 = Rp4.000,00.

20.

Anto meminjam uang di koperasi sebesar Rp6.000.000,00 dengan bunga 1,5% per bulan. Jika lama meminjam 10 bulan, besar angsuran yang harus dibayar setiap bulan adalah...

Lihat Pembahasan

Jawaban: C

Bunga per bulan = 1,5% x 6.000.000 = Rp90.000. Pokok angsuran per bulan = 6.000.000 / 10 = Rp600.000. Total Angsuran = Pokok + Bunga = 600.000 + 90.000 = Rp690.000,00.

21.

Jika harga beli Rp100.000,00 dan dijual Rp80.000,00, maka persentase kerugiannya adalah...

Lihat Pembahasan

Jawaban: B

Rugi = 100.000 - 80.000 = 20.000. %Rugi = (20.000 / 100.000) x 100% = 20%.

22.

Harga awal sebuah kemeja adalah Rp200.000,00. Toko memberikan diskon 50% + 20%. Harga akhir kemeja tersebut adalah...

Lihat Pembahasan

Jawaban: B

Diskon pertama 50%: Harga menjadi 50% x 200.000 = 100.000. Diskon kedua 20% dari harga baru: 20% x 100.000 = 20.000. Harga Akhir = 100.000 - 20.000 = Rp80.000,00. (Catatan: 50%+20% bukan 70%).

23.

Pedagang buah membeli 20 kg jeruk seharga Rp160.000,00. 1 kg jeruk busuk dan tidak bisa dijual. Sisanya dijual dengan harga Rp10.000,00 per kg. Berapa keuntungan pedagang?

Lihat Pembahasan

Jawaban: B

Jeruk yang bisa dijual = 20 - 1 = 19 kg. Total HJ = 19 kg x 10.000 = 190.000. Untung = 190.000 - 160.000 = Rp30.000,00.

24.

Pak Hasan menabung Rp10.000.000,00 di bank. Setelah 6 bulan uangnya menjadi Rp10.600.000,00. Suku bunga per tahun bank tersebut adalah...

Lihat Pembahasan

Jawaban: D

Bunga (6 bulan) = 600.000. Bunga 1 tahun = (12/6) x 600.000 = 1.200.000. %Bunga per tahun = (1.200.000 / 10.000.000) x 100% = 12%.

25.

Sebuah barang dijual Rp115.000,00 dan pedagang untung 15%. Besar keuntungan nominalnya adalah...

Lihat Pembahasan

Jawaban: B

Cari HB dulu. HB = (100 / 115) x 115.000 = 100.000. Untung = HJ - HB = 115.000 - 100.000 = Rp15.000,00.

B. Essay

Esai 1.

Seorang pedagang membeli 50 kg mangga dengan harga Rp400.000,00. Sebanyak 40 kg dijual dengan harga Rp10.000,00 per kg, dan sisanya dijual dengan harga Rp8.000,00 per kg. Tentukan apakah pedagang tersebut untung atau rugi, dan berapa persentasenya!

Lihat Jawaban Lengkap

Pembahasan:

1. Modal (HB) = Rp400.000 2. Penjualan 1: 40 kg x Rp10.000 = Rp400.000 3. Penjualan 2 (Sisa): 10 kg x Rp8.000 = Rp80.000 4. Total HJ = 400.000 + 80.000 = Rp480.000 5. Karena HJ > HB, pedagang UNTUNG. 6. Besar Untung = 480.000 - 400.000 = Rp80.000 7. %Untung = (80.000 / 400.000) x 100% = 20%.

Esai 2.

Ayah meminjam uang di bank sebesar Rp12.000.000,00 yang akan dilunasi dengan cara diangsur selama 10 bulan. Jika bank menetapkan bunga pinjaman sebesar 10% per tahun, hitunglah besar angsuran yang harus dibayar Ayah setiap bulannya!

Lihat Jawaban Lengkap

Pembahasan:

1. Pokok Pinjaman = Rp12.000.000 2. Bunga per tahun = 10% 3. Bunga untuk 10 bulan = (10/12) x 10% x 12.000.000 = (10/12) x 1.200.000 = Rp1.000.000 4. Total Hutang = Pokok + Bunga = 12.000.000 + 1.000.000 = Rp13.000.000 5. Angsuran per bulan = Total Hutang / 10 = 13.000.000 / 10 = Rp1.300.000,00.

Esai 3.

Sebuah toko elektronik memberikan diskon 15% untuk semua barang. Jika Budi membeli sebuah laptop dan membayar sebesar Rp4.250.000,00 setelah didiskon, berapakah harga awal laptop tersebut sebelum didiskon?

Lihat Jawaban Lengkap

Pembahasan:

1. Persentase harga setelah diskon = 100% - 15% = 85% 2. Harga Awal = (100 / 85) x Harga Bayar 3. Harga Awal = (100 / 85) x 4.250.000 4. Harga Awal = 100 x 50.000 = Rp5.000.000,00.

Esai 4.

Pak Ahmad membeli 2 karung gula pasir dengan berat masing-masing 50 kg dan tara 2%. Harga pembelian setiap karung adalah Rp400.000,00. Jika Pak Ahmad menjual gula pasir itu secara eceran dengan harga Rp10.000,00 per kg, berapakah total keuntungan yang diperolehnya?

Lihat Jawaban Lengkap

Pembahasan:

1. Total Modal = 2 karung x Rp400.000 = Rp800.000 2. Bruto total = 2 x 50 kg = 100 kg 3. Tara = 2% x 100 kg = 2 kg 4. Netto total = 100 kg - 2 kg = 98 kg 5. Total Penjualan (HJ) = 98 kg x Rp10.000 = Rp980.000 6. Untung = HJ - HB = 980.000 - 800.000 = Rp180.000,00.

Esai 5.

Ani menabung di bank dengan bunga tunggal 8% per tahun. Jika ia ingin jumlah tabungannya menjadi Rp2.160.000,00 dari modal awal Rp2.000.000,00, berapa bulan Ani harus menabung?

Lihat Jawaban Lengkap

Pembahasan:

1. Bunga yang diharapkan (Rp) = 2.160.000 - 2.000.000 = 160.000 2. Rumus: Bunga = (n/12) x %B x Modal 3. 160.000 = (n/12) x (8/100) x 2.000.000 4. 160.000 = (n/12) x 160.000 5. 160.000 / 160.000 = n/12 6. 1 = n/12, maka n = 12 bulan.

Esai 6.

Seorang karyawan menerima gaji Rp5.000.000,00 sebulan dengan penghasilan tidak kena pajak (PTKP) Rp1.500.000,00. Jika pajak penghasilan (PPh) sebesar 10%, hitunglah gaji bersih yang diterima karyawan tersebut dalam satu tahun!

Lihat Jawaban Lengkap

Pembahasan:

1. Penghasilan Kena Pajak (PKP) per bulan = 5.000.000 - 1.500.000 = 3.500.000 2. Pajak per bulan = 10% x 3.500.000 = 350.000 3. Gaji bersih per bulan = 5.000.000 - 350.000 = 4.650.000 4. Gaji bersih setahun = 4.650.000 x 12 = Rp55.800.000,00.

Esai 7.

Sebuah barang dibeli dengan harga Rp1.000.000,00. Pedagang ingin menjual barang tersebut dengan memberikan diskon 20% kepada pembeli, namun tetap ingin mendapatkan untung 20% dari harga beli. Berapakah harga label yang harus dipasang pada barang tersebut?

Lihat Jawaban Lengkap

Pembahasan:

1. Harga Jual (Netto) yang diinginkan = Modal + (20% x Modal) = 1.000.000 + 200.000 = 1.200.000. 2. Harga Jual ini adalah 80% dari Harga Label (karena diskon 20%). 3. Harga Label = (100 / 80) x Harga Jual Netto 4. Harga Label = (100 / 80) x 1.200.000 = 1,25 x 1.200.000 5. Harga Label = Rp1.500.000,00.

Esai 8.

Toko A dan Toko B menjual jenis jaket yang sama dengan harga Rp200.000,00. Toko A memberikan diskon tunggal 30%. Toko B memberikan diskon bertingkat 20% + 10%. Toko manakah yang lebih murah dan berapa selisih harganya?

Lihat Jawaban Lengkap

Pembahasan:

1. Toko A: Diskon 30% x 200.000 = 60.000. Bayar = 140.000. 2. Toko B: Diskon I = 20% x 200.000 = 40.000. Sisa = 160.000. Diskon II = 10% x 160.000 = 16.000. Total Diskon = 56.000. Bayar = 144.000. 3. Kesimpulan: Toko A lebih murah. 4. Selisih = 144.000 - 140.000 = Rp4.000,00.

Esai 9.

Pak Dedi membeli 10 peti telur dengan harga Rp180.000,00 per peti. Setiap peti memiliki bruto 20 kg dan tara 5%. Telur tersebut dijual eceran dengan harga Rp12.000,00 per kg. Hitunglah total keuntungan Pak Dedi!

Lihat Jawaban Lengkap

Pembahasan:

1. Total Modal = 10 x 180.000 = 1.800.000. 2. Total Bruto = 10 x 20 kg = 200 kg. 3. Total Tara = 5% x 200 kg = 10 kg. 4. Total Netto = 200 - 10 = 190 kg. 5. Total Penjualan = 190 kg x 12.000 = 2.280.000. 6. Keuntungan = 2.280.000 - 1.800.000 = Rp480.000,00.

Esai 10.

Seseorang meminjam uang di bank sebesar Rp10.000.000,00. Ia mengembalikan pinjaman tersebut dengan angsuran Rp1.100.000,00 per bulan selama 10 bulan. Hitunglah persentase bunga per tahun yang ditetapkan bank tersebut!

Lihat Jawaban Lengkap

Pembahasan:

1. Total yang dibayarkan = 1.100.000 x 10 = 11.000.000. 2. Total Bunga (10 bulan) = 11.000.000 - 10.000.000 = 1.000.000. 3. Rumus: Bunga = (n/12) x %B x Modal 1.000.000 = (10/12) x %B x 10.000.000 1.000.000 = (5/6) x 10.000.000 x %B 1.000.000 = 8.333.333 x %B %B = 1.000.000 / 8.333.333 = 0,12 = 12%. 4. Jadi, bunga per tahun adalah 12%.
